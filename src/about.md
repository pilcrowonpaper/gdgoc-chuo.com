# GDGoC Chuo University

[Google Developer Groups on Campus（GDG on Campus）](https://gdg.community.dev/) は、Google Developers のサポートを受けて運営される、大学単位の技術系学生コミュニティです。110 か国以上、1200 以上の拠点があります。このコミュニティは、テクノロジーに興味を持つ人のためのコミュニティです。

その中でも GDGoC Chuo University は、中央大学を拠点に活動するコミュニティとして、「テクノロジーの力で身近な課題を解決する」「技術力を向上させる」ことを目指しています。GDGoC Chuo University は、次の３つの部門から構成されています：

-   Technology 部門：技術を活用して課題解決に取り組み、メンバーのスキル向上を目指す。
-   Event 部門：イベントの企画・運営を通して、コミュニティを盛り上げ、つながりを深める。
-   Marketing 部門：活動内容を発信し、GDGoC Chuo University の魅力を広めていく。

現在は中央大学市ヶ谷田町キャンパス、後楽園キャンパス、茗荷谷キャンパスで活動しています。

## プロジェクト

-   Chuinfo：日々レポートに追われる中央大生のために、大学 Web サービスと連携する様々な便利な機能がついたアプリ。([iOS](https://apps.apple.com/jp/app/chuinfo-%E4%B8%AD%E5%A4%AE%E5%A4%A7%E7%94%9F%E5%90%91%E3%81%91%E9%9D%9E%E5%85%AC%E5%BC%8F%E3%82%A2%E3%83%97%E3%83%AA/id6670358906), [Android](https://play.google.com/store/apps/details?id=com.myml_12.chuinfo&pli=1))
-   [LAV（ラボ）](https://lav.gdgoc-chuo.com)：ウェブ上で動くレトロ風の仮想ゲーム機とゲーム制作ソフト。プログラミングやキャラクターデザイン、効果音の制作などをすべてひとつの場所で行え、作ったゲームをすぐに遊べます。2025年 iTLFest と白門祭（茗荷谷キャンパス）で行ったゲーム開発体験で使いました。（[GitHub](https://github.com/pilcrowonpaper/lav)）
-   [Puffin](https://github.com/pilcrowonpaper/lav/blob/main/documentation/puffin-jp.md)：初心者向けの自作言語。LAV のゲームのプログラミング言語として使われています。
